# 🤖 ClarifAI

ClarifAI is a lightweight, local-first AI-powered FAQ assistant. It uses mock embeddings and FAISS to semantically match user questions to pre-loaded FAQs, wrapped in a clean Streamlit UI.

---

## 🔧 Features
- Chat-style interface for asking FAQ questions
- Semantic search using FAISS (mocked for offline use)
- Streamlit-powered browser UI
- Clean design with branding and styled chat bubbles
- Sidebar with app info and suggested questions

---

## 🧱 Tech Stack
- Python 3.9+
- FAISS (vector search)
- Streamlit (frontend)
- Numpy (vector math)
- dotenv (local secrets)
